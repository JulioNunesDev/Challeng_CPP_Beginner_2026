# Trajetória C++

Este repositório documenta minha jornada de aprendizado em C++, começando do zero com 5 desafios práticos de nível fácil — do I/O básico até lógica de jogo orientada a objetos.

Cada desafio inclui:
- 📄 Código-fonte completo
- 🧠 Teoria e conceitos por trás da implementação
- 🐞 Bugs reais encontrados e corrigidos durante o desenvolvimento
- ✅ O que foi consolidado como aprendizado

## Progressão dos desafios

1. **Jogo de Adivinhação de Número** — I/O, loops, condicionais, `<random>`
2. **Lista de Tarefas com Persistência em Arquivo** — `struct`, `std::vector`, arquivos
3. **Calculadora com Menu e Funções** — modularização, `switch`/`enum`
4. **Sistema de Cadastro de Alunos (OOP)** — classes, encapsulamento
5. **Jogo da Velha no Console** — arrays 2D, lógica de jogo

A ordem importa: cada desafio constrói sobre o conceito anterior.
